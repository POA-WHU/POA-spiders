# POA-spiders
Web crawlers used in Public Opinions Analysis

## Tested Environment
> + Windows 10 Pro 64-bit 20H2
> + python 3.9.2

## Dependencies
```commandline
pip install beautifulsoup4
pip install requests
```
or just
```commandline
pip install -r requirements.txt
```
