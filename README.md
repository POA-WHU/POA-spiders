# spider_demo
A demo showing how to build a simple web spider.

## Tested Environment
> + python 3.7.9
> + pip 21.0.1

## Dependencies
```commandline
pip install beautifulsoup4
pip install requests
```
or just
```commandline
pip install -r requirements.txt
```
